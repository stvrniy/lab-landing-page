# Bachelor's Thesis Landing Page

## Development of Intelligent Educational Process Monitoring System

This is a landing page for my bachelor's thesis on the development of an intelligent educational process monitoring system using modern web technologies.

## Features

- **Semantic HTML5** - Proper document structure with accessibility in mind
- **Responsive Design** - Works on all devices (desktop, tablet, mobile)
- **SEO Optimized** - Meta tags, Open Graph, Schema.org markup
- **Multilingual Support** - Ukrainian and English versions
- **Accessibility** - ARIA attributes, proper heading structure, alt texts

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (Vanilla)
- SVG Icons

## Branching Strategy

This project follows **GitHub Flow** branching strategy:

- `main` - Production-ready code, always deployable
- Feature branches created from `main` for new features
- Pull requests for code review before merging

## Git Commits

The development process is documented through meaningful commit messages:

1. **Initial commit** - Project structure setup
2. **Basic HTML structure** - Semantic markup with meta tags
3. **CSS framework** - Normalize and base styles
4. **Main styles** - Complete styling with responsive design
5. **JavaScript functionality** - Navigation and interactions
6. **SEO optimization** - Meta tags and Schema.org
7. **Multilingual support** - English version
8. **Final touches** - Accessibility improvements
