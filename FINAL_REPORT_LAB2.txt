1. МЕТА РОБОТИ
================================================================================

Розробити односторінковий веб-сайт для представлення бакалаврської роботи з
дотриманням сучасних вимог до веб-розробки та використанням системи контролю
версій Git. Продемонструвати розуміння та практичне застосування стратегій
гілкування, принципів чистих комітів та ведення проектної документації.

================================================================================
2. ХІД ВИКОНАННЯ
================================================================================

2.1. ОБРАНА СТРАТЕГІЯ ГІЛКУВАННЯ: GitHub Flow

GitHub Flow — це проста та легковага стратегія гілкування, яка ідеально
підходить для веб-проектів з безперервним розгортанням.

Основні принципи:
- Гілка `main` — завжди готова до розгортання, містить стабільний код
- Feature-гілки — створюються для нових функцій з гілки `main`
- Pull Requests — обов'язковий код-рев'ю перед злиттям
- Негайне розгортання — після злиття в `main` код можна відразу деплоїти

2.2. ЕТАП 60%: БАЗОВА СТРУКТУРА

Створено:
- index.html — головна сторінка українською мовою
- en/index.html — англійська версія
- css/normalize.css — CSS reset для кросбраузерності
- css/styles.css — основні стилі (1000+ рядків)
- js/main.js — JavaScript для інтерактивності
- .gitignore — ігнорування непотрібних файлів

2.3. ЕТАП 74%: ДОСТУПНІСТЬ ТА SEO

Реалізовано:
- Семантична розмітка HTML5 (<header>, <main>, <section>, <footer>)
- ARIA-атрибути для доступності
- Альтернативні тексти для всіх зображень
- Заголовкова структура (h1-h6)
- Schema.org JSON-LD розмітка для Thesis
- Open Graph теги для соцмереж
- Twitter Cards метатеги

2.4. ЕТАП 82%: КОНТЕНТНЕ НАПОВНЕННЯ

Створено обов'язкові секції:
- Назва роботи (українською та англійською)
- Короткий опис (до 300 символів)
- Ключові слова
- Мета дослідження
- 5 основних завдань
- Очікувані результати (4 пункти)
- Контактна інформація

2.5. ЕТАП 100%: СТИЛІЗАЦІЯ ТА АДАПТИВНІСТЬ

Реалізовано:
- Адаптивний дизайн (mobile-first підхід)
- CSS Custom Properties (змінні)
- Брейкпоінти: 1024px, 768px, 480px
- Типографіка з Google Fonts
- SVG іконки для швидкого завантаження
- Плавний скрол та анімації
