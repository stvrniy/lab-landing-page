About the Thesis

This bachelor's thesis is dedicated to the development of an intelligent educational process monitoring system that automates the receipt and processing of data about class schedules, academic performance, and other important aspects of the educational process.

The main result of the work is a Telegram bot and web interface that provides students and teachers with convenient access to up-to-date information in real time.

Relevance of the Topic

Digitalization of Education

The modern educational process requires effective digital tools for managing the educational process and communication between participants.

Automation

Automation of routine processes saves time for teachers and students, focusing on main educational tasks.

Information Accessibility

Ensuring fast and convenient access to important educational information through popular messengers.

Goal and Tasks of the Research

Research Goal

Development of an intelligent educational process monitoring system based on a Telegram bot and web interface for automating the receipt and processing of educational process data.

Main Tasks

  • 1 Analyze existing solutions and determine system requirements
  • 2 Develop system architecture and select technology stack
  • 3 Implement module for university API integration
  • 4 Develop Telegram bot with extended functionality
  • 5 Create web interface for system administration

Research Methodology

Requirements Analysis

Using survey and questionnaire methods to identify user needs of the system.

Design

Using UML diagrams to visualize system architecture and data flows.

Development

Using Agile methodology with two-week sprints and regular demonstrations.

Testing

Applying unit, integration, and E2E testing to ensure quality.

Expected Results

Functional Telegram Bot

Full-featured bot with schedule viewing, announcements, and system communication

Web Admin Panel

Convenient interface for content management and usage statistics viewing

Integration API

Documented API for extension and integration with other systems

Technical Documentation

Complete documentation set: technical specification, user manual, API description
