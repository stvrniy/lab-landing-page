Про роботу

Дана бакалаврська робота присвячена розробці інтелектуальної системи моніторингу освітнього процесу, яка дозволяє автоматизувати отримання та обробку даних про розклад занять, академічну успішність та інші важливі аспекти навчального процесу.

Основним результатом роботи є Telegram-бот та веб-інтерфейс, що надають студентам та викладачам зручний доступ до актуальної інформації в режимі реального часу.

Актуальність теми

Цифровізація освіти

Сучасний освітній процес потребує ефективних цифрових інструментів для управління навчальним процесом та комунікації між учасниками.

Автоматизація

Автоматизація рутинних процесів дозволяє зекономити час викладачів та студентів, зосереджуючись на основних освітніх задачах.

Доступність інформації

Забезпечення швидкого та зручного доступу до важливої навчальної інформації через популярні месенджери.

Мета та завдання дослідження

Мета дослідження

Розробка інтелектуальної системи моніторингу освітнього процесу на базі Telegram-боту та веб-інтерфейсу для автоматизації отримання та обробки даних про навчальний процес.

Основні завдання

  • 1 Проаналізувати існуючі рішення та визначити вимоги до системи
  • 2 Розробити архітектуру системи та обрати технологічний стек
  • 3 Реалізувати модуль інтеграції з API університету
  • 4 Розробити Telegram-бот з розширеним функціоналом
  • 5 Створити веб-інтерфейс для адміністрування системи

Методологія дослідження

Аналіз вимог

Використання методів опитування та анкетування для визначення потреб користувачів системи.

Проектування

Застосування UML-діаграм для візуалізації архітектури системи та потоків даних.

Розробка

Використання Agile-методології з двухтижневими спринтами та регулярними демонстраціями.

Тестування

Застосування модульного, інтеграційного та E2E тестування для забезпечення якості.

Очікувані результати

Функціональний Telegram-бот

Повноцінний бот з можливістю перегляду розкладу, отримання анонсів та комунікації з системою

Веб-панель адміністрування

Зручний інтерфейс для керування контентом та перегляду статистики використання

API інтеграції
